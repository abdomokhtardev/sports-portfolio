# Client A

مشروع React + Vite لإنشاء موقع احترافي لشخصية مترجم/مقدمة خدمات ترجمة، مع صفحة عرض تقديمي ولوحة إدارة بسيطة متكاملة مع Supabase.

## نظرة عامة

هذا المشروع مصمم لعرض الهوية المهنية والخدمات والمهارات والموارد البصرية للمترجم أو الخبير في مجال الترجمة بشكل أنيق وسهل الإدارة. يتضمن:

- صفحة رئيسية احترافية
- قسم عن المشروع/الشخص
- أقسام الخبرة والتجارب
- قسم شهادات/تقييمات
- عرض فيديوهات مهنية
- صفحة تسجيل دخول ولوحة إدارة لإدارة المحتوى

## التقنيات المستخدمة

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- Supabase JavaScript SDK

## هيكل المشروع

- src/components: المكونات الرئيسية للواجهة
- src/data: البيانات الثابتة الخاصة بالمحتوى
- src/lib: إعدادات Supabase والخدمات
- src/hooks: الهوكس المخصصة
- supabase: ملفات SQL الخاصة بإعداد قاعدة البيانات

## المتطلبات الأساسية

- Node.js 18 أو أحدث
- npm 9 أو أحدث

## طريقة التثبيت

1. افتح المجلد في الطرفية
2. شغل الأمر التالي:

```bash
npm install
```

## إعداد المتغيرات البيئية

أنشئ ملف .env في جذر المشروع وأضف المتغيرات التالية:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

> يجب ألا تُرفع هذه القيم إلى Git، وقد تم تجاهلها من خلال ملف .gitignore.

## تشغيل المشروع

```bash
npm run dev
```

سيتم تشغيل التطبيق على العنوان الافتراضي لـ Vite عادةً على:

```text
http://localhost:5173
```

## بناء المشروع للإنتاج

```bash
npm run build
```
